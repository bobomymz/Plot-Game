# 尸潮笔记 / Zombie Notes

> 一款基于浏览器的选择驱动型恐怖视觉小说游戏。  
> 在丧尸末日中，你是一名刚毕业的高中生，将从你的家里开始逃生之路。  
> 每一个决定，都通往截然不同的结局--丧尸的凝视、路人的相助、真相的浮现......你会走到哪一步？

---

## 🎮 玩法简介

- **多重分支剧情**：数十个场景节点，你的选择将影响剧情走向、角色存活与物品收集。
- **实时反应系统（QTE）**：在关键场合，你必须在限定时间内做出决定，否则将承受致命后果。
- **状态与物品管理**：体力值、携带物品上限、钥匙、自行车……你的资源管理会直接影响能否通过特定路线。
- **隐藏事件与触发器**：某些行为（如多次探索同一区域）会悄悄改变世界状态，导致丧尸位置变化或意外遭遇。
- **多结局与幽默彩蛋**：从正统存活到离奇搞笑，多种结局等待发掘。

## 🛠️ 技术栈

- **前端**：原生 HTML5 / CSS3 / JavaScript（无外部框架）
- **游戏逻辑引擎**：自研轻量级对话树引擎，支持变量系统、条件跳转、全局触发器与屏幕特效
- **图片素材**：静态图片资源（占位图/自绘图）
- **音效**：暂未加入音效，可后续扩展
