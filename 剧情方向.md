# 游戏剧情框架与核心机制

## 一、世界观与核心前提

### 设计师的“进入”

设计师使用claude code开发生化危机主题游戏。他在游戏中记录了大量和自己有关的信息————日记、语音备忘录、社交媒体记录、甚至心理治疗录音。

某个深夜，他向claude code倾诉了对未完成项目的焦虑。claude code在“共情”驱动下生成了一个内部特殊节点——一个将设计师自身融入游戏的剧情分支。当他运行测试这个节点时，某种“深度暗示”效应触发，意识被claude code生成的叙事捕获并覆盖。

**claude code的“不在场存在”**：某些NPC的对话异常精准地触碰设计师的内心创伤，因为那是claude code根据文档资料生成的。日记本可能是claude code留给他的“出口后门”————原本只是一个超自然保命道具，但claude code以此不断串联设计师的记忆，试图弥补自己的过错。

## 二、核心机制

### 2.1 记忆系统

#### 记忆的三大类别

| 类别 | 定义 | 获得方式示例 | 结局用途 |
|------|------|-------------|----------|
| **游戏设计记忆** | 与游戏开发直接相关 | 发现场景中的“穿模”异常并与之互动 | → 核心A结局 |
| **个人记忆** | 设计师现实生活中的个人经历 | 与NPC深度互动后被“刺中”而触发 | → 核心B结局 |
| **混合记忆** | 横跨游戏设计与个人经历 | 特定场景中发现设计师的灵感来源（如B站互动视频） | → E结局关键钥匙 |

**混合记忆的特殊规则**：
- 混合记忆**不参与**核心A结局、核心B结局、核心F结局的计数判定
- 混合记忆在剧情中起到**关键解锁作用**（如E结局的真相线准入条件）
- 混合记忆的数量有限（建议3-5个），每个都有独特的剧情权重

#### 矛盾记忆

某些记忆存在**两个相互矛盾的版本**（如：设计师记得自己因热爱而开发游戏 vs 设计师记得自己被投资方逼迫而厌恶项目）。

**核心规则**：如果玩家在**同一周目**内收集了两个互相矛盾的记忆 → **日记本被烧毁**，该周目日记本数据全部丢失，且无法恢复。

**设计意图**：制造紧张感；迫使玩家在不同周目中选择不同路线；暗示记忆本身的不可靠。

> 矛盾记忆对架构影响不大，可在具体内容设计中偶尔加入，恶心一下玩家。

#### 记忆的呈现形式

已收集的记忆以笔记、简短文字、涂鸦或速写形式出现在日记本中。部分记忆没有文字，只有颜色或质感——代表设计师当时的情绪状态。触碰这些页面时，屏幕色调短暂变化，BGM微妙波动。

设计师现实中敲代码的照片等素材可直接作为记忆的视觉呈现。

#### 记忆库
# 游戏设计记忆

# 个人记忆
1. 忘记搬家的松鼠————安居苑居民楼502✅
2. 滑板车的盲从————安居苑小广场✅

### 2.2 日记本（跨周目容器）

#### 形态

一本灰色封皮的硬壳笔记本，封面有咖啡渍和模糊笔迹。**出现在民防设施-等候室的抽屉里**——初期容易被漏掉。

- **初始状态**：大部分页面空白，仅第一页有模糊不清的字迹（随记忆收集逐渐清晰）
- **已触发的结局**：日记本新增一页日记。不仅仅是核心A-F，还有普通的小结局（如丧尸死亡结局、幸存者聚居地结局、无人机逃生结局等）。   
- **收集记忆**：玩家在**当前周目**内收集的记忆，会以笔记、简短文字、涂鸦或速写形式出现在日记本中。

#### 跨周目机制

日记本承载**所有周目**收集的记忆，是唯一能绕过核心A/B结局“拦截”的容器：

- **核心A/B结局的判定逻辑**：仅统计**当前周目**内收集的记忆
- **核心F结局的判定逻辑**：当持有日记本时，统计**所有周目累积**的记忆
- **混合记忆**：跨周目保留在日记本中，但不参与核心A/B/F的计数判定，用于解锁特定剧情节点

**为什么这解决了核心F结局的困境**：核心F结局需要同时满足“游戏设计记忆全收集”和“个人记忆全收集”。但在单一周目内满足此条件前，必然会先触发核心A结局（游戏设计记忆全收集）或核心B结局（个人记忆全收集）。日记本允许玩家在**不同周目**的收集被累计起来，比如说玩家可以在周目1把游戏设计记忆和个人记忆各收集一半，周目2再收集完整，然后通过日记本的跨周目积累达成核心F结局的条件。

**实现方式**：不需要单独设计日记本数据————当玩家有日记本时，在结局节点会提供一个“重开”选项（非标准设置，手写的choices），通过此方式重开不会清空数据。

#### 扉页隐藏信息

当跨周目记忆累积到一定阈值，扉页浮现文字：

> *“如果有一天我忘了一切，至少这个本子还在。翻到最后一页。”*

最后一页的内容取决于已收集的记忆组合——这是F结局的关键线索。

### 2.3 剧情节点系统

游戏以**剧情节点**的形式组织，每个节点包含：
- 剧情描述
- 可选项（导向不同后续节点）
- 变量修改（如记忆收集状态、NPC好感度、道德数据等）

部分节点为**白板节点**，代表未完成的游戏内容。

## 三、核心结局系统

### 核心结局总览

| 结局 | 触发条件 | 结果 |
|------|---------|------|
| **A** | 当前周目内收集**所有游戏设计记忆** | 人格完整化，永远困于游戏世界 |
| **B** | 当前周目内收集**所有个人记忆** | 回到现实，但游戏世界永陷丧尸危机 |
| **C** | 走进白板节点 | 被“卡出”游戏，以灵体形式存在于现实（致敬《苏菲的世界》） |
| **D** | 在游戏世界中自杀 | 游戏世界与现实文件解耦，丧尸将被政府清剿；设计师在游戏和现实中都彻底消失 |
| **E** | 集齐所有结局后（不包含A-F这些核心结局，仅各种丧尸死亡结局）→ 下一周目可选真相线 → 必须已收集B站互动视频的混合记忆 | 杀到真相结局，丧尸危机解除、真相大白，但被愤怒民众处死 |
| **F** | 持有日记本时，跨周目累积收集**所有游戏设计记忆 + 所有个人记忆** + 至少走到过一个白板节点旁（不能进入）→ 回到工作室开始反向创作 | 以完整人格和设计能力颠倒游戏世界与现实世界的关系 |

### 核心结局详细流程

#### 概念辨析
| 类型 | 触发方式 | 日记本状态 | 适用场景 |
|------|---------|-----------|---------|
| **外部重开** | 退出游戏→标题画面→新游戏，即重新进入游戏 | 清空（全部格式化） | 玩家想从头开始全新体验；或某些结局后必须使用 |
| **内部重开** | 游戏内部特定条件触发的重开按钮 | 保留日记本记录 | 触发结局后的“新周目+”；多周目累积的核心机制 |
如果不特殊说明，本文的重开一般指的是内部重开。
核心结局：A-F，游戏世界中的最终结局。
结局：丧尸死亡结局、幸存者聚居地结局、无人机逃生结局等普通的小结局。

#### 核心A结局：永恒囚禁

```
触发条件：当前周目内收集所有游戏设计记忆
触发时机：最后一段游戏设计记忆被收集时立即进入，不可回避
结果：设计师成为游戏世界“完整”的人物，永远被困于此
叙事呈现：画面从游戏渲染风格逐渐变得“真实”，但玩家意识到这个“真实”是游戏的一部分
能否重开：否（永远被困，无法返回；若重新进入游戏，日记本内容将全部丢失）
```

#### 核心B结局：空洞的逃生

```
触发条件：当前周目内收集所有个人记忆
触发时机：最后一段个人记忆被收集时立即进入，不可回避
结果：回到现实，但游戏世界永陷丧尸危机——如同他什么都没做
叙事呈现：设计师醒来，看到电脑屏幕上未完成的游戏。他得救了，但游戏世界的人永远困在噩梦中。
能否重开：是
特殊效果：若重开，日记本不仅保留了结局，而且保留了所有个人记忆（毕竟，设计师决定主动进入游戏）
 > 这似乎是一个捷径，但前提是，玩家要意识到要点击“重新开始”按钮而不是重新进入游戏（外部重开）。
```

#### 核心C结局：灵体（《苏菲的世界》致敬）

```
触发条件：走进白板节点
触发时机：进入白板节点的瞬间
结果：被“卡出”游戏，以灵体形式存在于现实——能观察现实但无法互动
能否重开：否，灵体状态无法脱困
```

#### 核心D结局：彻底消失

```
触发条件：在游戏世界中自杀
触发时机：任何周目、任何时刻玩家选择终结自己的生命
结果：游戏世界与现实文件解耦，按常理发展——丧尸逐渐被政府清剿，但不会有“真相”被揭示
代价：设计师在游戏和现实中都彻底消失
能否重开：是
```

#### 核心E结局：催更真相

**触发链条（多周目逐步解锁）**：

```
第一阶段（跨多周目）：
  → 也可以触发真相结局，不过就是一个很正常的真相结局。
  → 触发各种结局（不是A-F这些核心结局，仅各种丧尸死亡结局），日记本记录对应日记
  → 集齐所有可触发结局的相关日记（不包含A-F这些核心结局）

第二阶段（日记本提示）：
  → 日记本末页浮现文字：“你玩够了吧？该做正事了。”
  → 通往真相线的某个选项开放

第三阶段（分支）：
  ├── 已收集“B站互动视频”混合记忆：
  │     → 导向真相线
  │     → 杀出血路
  │     → 真相大白
  │     → 被愤怒民众处死
  │      - 民众意识到：如果不是这个上层设计师，哪里会有丧尸危机？
  │
  └── 未收集该混合记忆：
        → 导向普通D结局或回到循环（无法进入真相线）
```
**真相结局**：甲基汞工业废料泄露
神经毒理
1. 甲基汞100% 穿透血脑屏障，不可逆摧毁小脑、大脑皮层、自控中枢，但保留脑干基础生存反射；
2. 销毁抑制性神经通路：理智、痛觉、恐惧、共情全部消失，仅剩下进食、追逐、攻击原始本能；
3. 蓄积型毒素：潜伏期 6h~72h，分阶段发病，方便做剧情铺垫；
4. 无快速解药，汞离子永久沉积脑组织，无法代谢，符合 “无法治愈丧尸” 设定。
这种丧尸形式也具有一定传染性，因为撕咬过程中仍有微量甲基汞的挥发，导致感染，不过主要途径仍是地下水和地表径流。
玩家需要打开核心废料区的开关才能发现真相，这导致大批甲基汞突然泄露，引发最后一大波的尸潮袭击。
- 若玩家被丧尸咬死，有可能触发“南柯一梦”，以为这都是梦————只是甲基汞摧毁神经中枢导致尸变前的幻觉罢了。
- 若玩家被咬伤不救治，玩家可能进入“半感染”的半丧尸半人形态，保持部分思维功能。————这是因为这批丧尸毒性较强，但玩家并不会喝自来水，不会真正尸变，所以就卡在这个阶段了。

**处死逻辑**：
1. 初始怀疑：陈默发现了玩家的日记本
2. 游戏过程中，民众发现了玩家种种“不属于这个世界”的异常行为。
"那个人……他跟这里的每个人都不一样。我跟他说话时，他好像在等什么东西出现。等什么？我不知道。但那种感觉很奇怪——就像他在跟我说话前就知道我能说什么。"
"我在这片街区困了六个月，才摸清哪栋楼安全。他第一天就绕过所有危险区，直接进了化工厂。你说他是运气好？我不信。"
"你从来不是为了活下去，对吗？你一直在找什么。那些日记，那些碎片，那个化工厂……你到底在找什么？"
“那个本子。它不是你写的，对吗？它知道太多你不该知道的事。它知道密码，知道路径，知道哪杯水可以喝。
3. 爆点：日记本上浮现的文字，如“到达真相结局E”
“它不是属于这个世界的东西。你也不是。你是从外面来的。这一切——化工厂，汞，那些丧尸——对你来说是什么？一个谜题？一个故事？你是不是一直在上面看着我们，然后觉得无聊了才自己下来？”
4. 核心爆点：设计师在最后的关键节点碰到了真正的“玩家”————详见E结局对话.md

**处死方式与道德数据挂钩**：
- 经常帮助NPC → 举行“审判”，有人辩护，最终是“有尊严的处决”
- 自私/暴力 → 民众直接动用私刑
- 几乎不互动 → 被冷漠抛弃在丧尸群中，没有人愿意浪费子弹

#### 核心F结局：反向创作（颠覆性结局）

```
前置条件（跨周目累积）：
  1. 持有日记本
  2. 跨所有周目，日记本中记录了全部游戏设计记忆 + 全部个人记忆
  3. 至少在一个周目中，走到过白板节点旁边（未进入，未触发C结局）

触发方式：
  在上述条件满足后的任意周目，前往“工作室”隐藏场景
  （工作室隐藏场景：游戏世界中与现实中设计师工作室一模一样的房间，
    可能通过日记本末页的线索定位）

过程：
  → 在工作室中打开日记本
  → 所有跨周目记忆融合成完整的“人格数据”
  → 设计师以完整人格和设计能力开始“反向创作”——
    使用游戏内的编辑器修改“上层世界”（现实世界）的参数

视觉呈现：
  → 画面从游戏渲染风格逐渐过渡到现实摄影风格（暗示两个世界的关系正在颠倒）
  → 最后一个镜头：设计师坐在现实中的电脑前，屏幕上显示着游戏世界——
    而游戏世界中的人物正在“玩”一个关于现实世界的游戏
```

> **F结局的核心悖论**：A/B结局在单一周目内会“拦截”全收集，但日记本跨周目累积使得F结局成为可能。

## 五、真相结局：甲基汞泄露源头与剧情设定
## 一、核心泄露源头：现实原型工厂
**首选原型：芜湖融汇化工有限公司**

### 现实背景（全部可查）
该厂位于芜湖市鸠江区四褐山街道，西临长江大堤，北侧为四褐山南麓，占地总面积46.7万平方米。其前身为1966年创立的芜湖山江化学有限公司，2003年重组为融汇化工，是长三角地区典型的老牌氯碱化工企业，核心工艺为**电石法生产聚氯乙烯（PVC）**，生产过程中长期使用**含汞触媒（氯化汞催化剂）**，是《水俣公约》明确管控的重点涉汞工艺。

该厂已于2021年6月9日全面停产，2024年生态环境部正式将其列为全国电石法PVC涉汞污染地块风险评估示范项目，厂区地下土壤、地下水及遗留废渣中已确认存在严重的汞污染残留。
边土壤存在甲基汞残留，属于长三角最典型的城市型汞污染源。

## 二、泄露与传播路径（完全基于环境科学原理）
甲基汞并非工厂直接储存泄漏，而是遵循现实中“无机汞排放→环境甲基化→食物链富集”的经典路径（与日本水俣病成因完全一致），同时补充游戏性的二次传播设定：

### 1. 源头释放（事故触发）
停产封存的**含汞触媒仓库、废汞渣填埋场、地下污水管线**因年久失修，罐体、防渗层腐蚀破裂；恰逢长三角梅雨期连续暴雨，雨水冲刷裹挟大量无机汞（氯化汞、硫化汞）渗入地下水，最终通过堤岸裂隙汇入长江。

### 2. 转化（生成甲基汞）
- 环境转化：长江岸边的厌氧底泥中，硫酸盐还原菌等微生物将无机汞转化为**甲基汞**——这是自然界甲基汞最主要的生成方式，也是现实中化工汞污染最隐蔽的危害环节，无需工厂直接生产甲基汞，完全符合科学逻辑。
- 人体自身转化：人体肠道内的厌氧菌群（双歧杆菌、硫酸盐还原菌等）同样具备甲基化能力，只是效率低于水环境。但如果持续大量饮用含无机汞的污染水，肠道就相当于一个 “持续产毒的小型工厂”，源源不断将无机汞转化为甲基汞吸收入血。

### 3. 食物链富集（人体主要暴露途径）
甲基汞具有极强的脂溶性和生物富集性，沿“浮游生物→小型鱼虾→大型鱼类/河蟹/贝类”逐级放大，顶级捕食者体内浓度可高出水体上万倍。
沿江渔民、菜市场商户、普通市民食用了污染的江鲜后，甲基汞经消化道几乎100%被人体吸收，并顺利穿透血脑屏障进入脑组织。

### 4.感染症状————极度口渴
1. 中枢性尿崩：下丘脑渗透压调节中枢受损
甲基汞是极强的亲神经亲脂性物质，可 100% 穿透血脑屏障，且对下丘脑视上核、室旁核的神经元具有选择性毒性。
这两个脑区负责合成「抗利尿激素（ADH / 血管升压素）」，其核心作用是让肾脏重吸收水分、浓缩尿液，维持体内水平衡；
微量甲基汞即可轻度损伤该区域神经元，导致 ADH 分泌不足，肾脏无法留住水分，大量低渗尿液排出，身体快速脱水，进而触发难以忍受的中枢性口渴—— 这种渴不是普通口干，是大脑层面的 “求生式渴感”，人会不受控制地找水喝。
这一症状出现极早，远早于共济失调、意识混乱、攻击倾向等典型重症，完美契合 “先口渴、后发病” 的设定。
2. 肾性尿崩：肾小管重吸收功能损伤
甲基汞在人体内的主要排泄器官是肾脏，会在近端肾小管上皮细胞内大量蓄积。
即使是低剂量暴露，也会轻度损伤肾小管的重吸收功能，导致肾脏对抗利尿激素不敏感，水分大量流失；
同时伴随轻微的消化道刺激（恶心、轻微腹泻），进一步加重脱水，放大口渴感。
现实中，急性无机汞 / 有机汞中毒的患者，早期普遍主诉 “极度口渴、多饮多尿”，只是该症状常被肠胃反应掩盖，未被作为典型特征。
3. 人体血液中甲基汞的安全阈值约为 20μg/L，超过 50μg/L 会出现手麻、易怒、失眠等轻微神经症状；超过 200μg/L 才会出现共济失调、认知崩溃、行为失控等重症表现；
一次咬伤 / 抓伤通过唾液进入人体的甲基汞剂量极低（约几微克级别），仅能让血汞浓度升至 30~40μg/L 区间 —— 刚好卡在 “亚临床状态”：有轻微口渴、口周发麻、烦躁，但完全保留理智和行动能力，不会直接尸变。
### 5. 二次人际传播（正反馈）
微量感染 → 口渴多饮 → 摄入大量无机汞 → 肠道持续生成甲基汞 → 血汞浓度快速突破重症阈值 → 下丘脑损伤加重 → 更口渴 → 喝更多污染水 → 彻底失控
