# 尸潮安全屋设计文档（v2）

## 核心机制

- **强制截止时间 DDL**：这是每个区域的属性，超出此时间后该区域所有安全屋将不再安全
- **触发时机**：当游戏内时间到达 19:00（hh >= 19）时，全局触发器强制跳转到"天黑必须过夜"场景
- **天黑后玩家必须在当前区域选择一个安全屋过夜，没有"跳过"或"返回"选项**

## 各区域安全屋

### 1. 初始小区（理论上不会触发夜间的区域）
- 无安全屋
- 兜底：随便出几个选项，然后跳到"被尸潮群殴"

### 2. 周边社区（DDL：Day 3 00:00）

| 安全屋 | 解锁条件 | 醒来地点 | 备注 |
|--------|---------|---------|------|
| 安盛街理发店 | 始终可用 | 理发店-店内 | Day 2 有特殊流浪结局 |
| 全家员工通道 | 员工丧尸已被清除 (`FamilymartHasZombie == false`) | 小区东门-整装待发 | 在全家内部选"外出"会死 |
| 小超市地下室 | 有手电筒 (`hasTorch`) | 三林路 | 没手电筒会摔死 |
| 安居苑空房 | 始终可用 | 三林安居苑-小区内部 | 需要先通过变异猫/入口 |

**兜底路线（始终可用）：**
- 露宿街头：体力 -2，`hurtByZombie = true`，随机在周边社区某处醒来
- DDL 之后（Day 3 00:00+）露宿街头 = 死亡

### 3. 高架
- 无安全屋
- 如果有车（`hasCar`）：可在车内过夜，体力 -1
- 否则露宿街头 = 死亡

### 4. 未来区域（预留）
- 迪士尼、临港、郊区 → 后续设计

## currentPlace 特化逻辑

```
if (currentPlace === "理发店")
  → 选项：就睡这里 / 冒险出去找别的
elif (currentArea === "周边社区")
  → 选项：去理发店 / 去全家 / 去小超市地下室 / 露宿街头
```

### 理发店 Day 2 特殊流浪结局
若玩家 Day 1 睡在理发店，Day 2 夜晚选择"离开四处游荡"：
- 奇迹幸存，但 `hurtByZombie = true`
- 跳转到"杨高南路高架"

## 实现方式

- 使用 `_globalTriggers` 检测 `hh >= 19`，最高优先级强制跳转
- 夜间场景无"退出"选项，只能选择安全屋或兜底
- 每个安全屋场景推进时间到次日 7:00
- 利用 `currentPlace` + `showCondition` 过滤可用选项
