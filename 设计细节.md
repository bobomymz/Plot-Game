## 一、皮肤颜色变化的科学解释

甲基汞通过血脑屏障后，皮肤颜色的变化本质上是神经-内分泌-循环系统连锁崩溃的外在表现，可以用来区分感染阶段：

阶段色：按病程划分

| 阶段 | 颜色 | 生理机制 | 科学依据 |
| ---- | ---- | -------- | -------- |
| 潜伏期（刚被咬/轻度暴露） | 面部潮红、皮肤油腻 | 汞中毒引起自主神经紊乱→血管舒张失调→交感神经过度激活→皮脂腺异常分泌 | 汞中毒的"红斑-多汗综合征"，历史上汞矿工人常见 |
| 进展期（血汞 50-100μg/L） | 灰白/蜡黄 | ① 汞蓄积肾脏→肾小管坏死→促红细胞生成素减少→肾性贫血→面色苍白<br>② 肝脏解毒负荷过大→轻度黄疸（眼白先黄） | 慢性汞中毒的典型"铅灰面色"，结合肾衰导致的贫血面容 |
| 重症期（血汞 >200μg/L） | 青紫（紫绀） | 下丘脑体温调节中枢被摧毁→体温随环境波动→外周血管极端收缩→末端组织缺氧→口唇/指尖紫绀 | 甲基汞导致的呼吸中枢抑制 + 外周循环衰竭 |
| 终末期（完全尸变） | 暗紫/褐绿 | ① 组织缺氧严重→血红蛋白分解→皮下淤血呈紫黑色<br>② 皮肤屏障破坏→继发产气菌感染→坏死组织呈现绿褐色<br>③ 汞抑制微生物→腐败速度反而慢于正常尸体 → 呈现"干尸化"的深褐色 | 这与常规丧尸的"快速腐烂"不同，用汞的抗菌特性解释了为什么丧尸尸体不快速烂掉 |

一个可用的视觉设计

感染程度 ↑
          褐绿/干尸化（终末）
         ↗
    暗紫发绀（重症）
   ↗
灰白蜡黄（进展）
   ↘
面部潮红（潜伏期）

进阶设定：汞沉积于皮肤真皮层，皮肤暴露在日光下会产生异常金属光泽（类似银汞合金的灰蓝色调），这在夜晚月光下特别明显——可以解释为什么夜间更难躲避丧尸。

## 二、"特殊能力"的科学解释

你的设定里丧尸是有机汞中毒+二次传播，这意味着所有"特殊能力"都是正常人类生理功能被破坏后产生的病理性副产品，而不是"超能力"。以下每个能力都有可查的毒理学文献支撑：

1. "狂暴之力"——痛觉消失 + 肌肉自我保护解除

现象：丧尸力量异常大，能撞开门、拽断自己的肢体依然前进。

科学解释：
- 甲基汞选择性摧毁丘脑腹后外侧核（痛觉中继站）和前扣带皮层（疼痛的情绪体验中枢）→ 完全痛觉缺失
- 正常人的肌肉收缩受高尔基腱器官限制（一个保护机制，防止肌肉撕裂自身），这种抑制信号由脊髓上中枢调控。高级中枢被摧毁后，肌肉可以100%发力而不是正常的30-40%
- 代价：每一次用力都在撕裂自己的肌腱和肌纤维——所以丧尸"力大无穷但动作不协调"

2. "不眠不休"——疲劳感缺失

现象：丧尸持续行走/追逐，不会停下来休息。

科学解释：
- 小脑是甲基汞最优先的攻击靶点（水俣病的标志性病理改变就是小脑颗粒细胞大量死亡）
- 小脑不仅管协调，还参与**"中枢疲劳信号"的生成**。小脑损坏后，大脑无法感知肌肉的代谢负担
- 同时，基底节受损→多巴胺能运动通路的抑制解除→持续的不自主运动冲动
- 结果：丧尸实质上处于一种病理性运动亢进状态（类似亨廷顿舞蹈症的失控运动，但方向是追逐捕食）
- 真实代价：肌肉持续无氧代谢→乳酸堆积→48小时后肌纤维溶解→肾功能衰竭→"丧尸也会瘫倒"（但能拖行）

3. "夜间活跃"——视交叉上核破坏

现象：白天反应迟钝，夜间异常敏锐。

科学解释：
- 甲基汞蓄积于视交叉上核（SCN，人体的主生物钟）→ 昼夜节律完全崩解
- 但更重要的是：视网膜光感受器受损模式特殊
  - 视锥细胞（明视觉、色觉）对汞更敏感，优先死亡 → 白天视力模糊
  - 视杆细胞（暗视觉）相对耐受 → 夜间视觉保留
  - 瞳孔括约肌被汞破坏 → 瞳孔固定散大（始终保持黑暗环境的最大进光量）
- 结果：丧尸处于永久暗适应状态，白天畏光眯眼，夜晚瞳孔全开异常敏锐

4. "趋水定向"（这可能是你设定中最独特的能力）

现象：丧尸会自发向水体/潮湿区域聚集，形成沿河岸、排水系统的移动路径。

科学解释：
你已经写了极度口渴（中枢性尿崩+肾性尿崩），可以再深入一步：

- 下丘脑渗透压感受器永久激活：正常人的抗利尿激素神经元受渗透压调节，水合充足时停止发放信号。汞中毒后，这些神经元的抑制性突触被选择性破坏，结果渗透压感受器处于永久性兴奋状态
- 大脑将"严重脱水"的虚假信号持续投射到意识层面（虽然已经没有意识可言）→ 一种铭刻在脑干反射层面的驱动力
- 这驱使丧尸朝任何高湿度区域移动：河岸、下水道、雨后洼地、甚至人类的呼吸（呼出气体含饱和水蒸气）
- 游戏性推导：尸潮移动路线可预测（沿水系），玩家可以利用这一点规划安全路线

5. "同步化尸潮"——而非蜂群思维

现象：大量丧尸的行为出奇一致，看起来像有组织。

科学解释（注意—这不是心灵感应，更符合你的硬科幻设定）：

- 甲基汞摧毁大脑皮层的抑制性GABA能中间神经元（这类神经元对汞离子极度敏感）
- 抑制解除 → 大脑进入过度兴奋状态 → 远处的丧尸能听到同一个声音来源并同步做出反应
- 这不是"共享意识"，而是：个体间因相同的神经损伤模式，对同一刺激产生相同的反射行为
- 类似于一群没有大脑皮层的动物——风声、枪响、某个人类的脚步声会在同一瞬间触发所有丧尸的朝向反射
- 游戏应用：如果一个丧尸开始朝某个方向跑，附近的丧尸几乎会同一瞬间也朝那个方向跑，看起来像"命令"，实则是共同的释放反射

6. "垂死喉音"——发声器官的病理性改变

现象：丧尸发出持续的低沉喉音/咯咯声，而非咆哮。

科学解释：
- 喉返神经（控制声带）对甲基汞高度敏感 → 声带麻痹 → 无法主动发声
- 但呼吸肌（膈肌、肋间肌）不受控制地痉挛 → 气流强行通过麻痹的声带 → 产生不自主的喉鸣音
- 同时，下咽部唾液积聚（因吞咽反射消失）+ 呼吸道的黏液 → 气流通过时产生 "湿性咯咯声"（死亡喉音）
- 定位价值：玩家可以在黑暗环境通过声音判断丧尸的位置和密度

## 三、用"汞沉积量"统一所有"特殊能力"

```
mercuryLoad: 0,   // 0-100, 隐藏（作为 gameState 中的一个隐藏变量）
```

- 血液中的甲基汞 > 50 μg/L → 开始出现轻微手麻、口渴（玩家能感知的线索）
- 每被咬一次/喝一次污染水 → mercuryLoad +10~20
- mercuryLoad 20-40 → 皮肤灰白、瞳孔散大、疲劳感降低
- mercuryLoad 40-70 → 痛觉消失、力量异常、夜间视力增强
- mercuryLoad 70+ → 完全失去理智，游戏判定"尸变"

## 四、毒气型丧尸——肠源性汞挥发综合征

### 4.1 现有设计中的矛盾

目前毒气丧尸在游戏中出现了三次，但缺乏统一的科学解释：

| 场景 | 现有描述 | 待解释的问题 |
|------|---------|-------------|
| **五金店** | 管道破裂冒出白色气体，丧尸"躲在毒气里" | 气体是管道泄漏还是丧尸发出的？ |
| **初遇毒气型丧尸（民防设施通道）** | 叫"毒气型"，但未实际与毒气互动 | 为什么叫毒气型？它特殊在哪？ |
| **民防设施-进风机房-失败** | 闻到刺鼻气味，头晕目眩，绿色眼睛靠近 | 气体成分是什么？为什么防毒面具有效？ |

**核心问题**：五金店的气体消杀是环境灾害（管道泄漏），进风机房的气体是丧尸自身发出的。两者需要统一的解释——或者明确区分。

### 4.2 机制设计——肠源性汞挥发综合征（Enteric Mercury Volatilization, EMV）

#### 化学路径

从甲基汞中毒到产气的生化链：

```
甲基汞 (CH₃Hg⁺)  →  [肝脏脱甲基]  → 无机汞 (Hg²⁺)
     ↓                                         ↓
→ 胆汁排泄入肠道                          → 随血流分布全身
     ↓                                         ↓
[肠道细菌作用]  →  元素汞蒸气 (Hg⁰)       → 肾排泄（部分被重吸收 → 再入胆汁 → 回到肠道）
```

**关键反应**——肠道中的硫酸盐还原菌（主要是 *Desulfovibrio* 属）在无氧条件下进行的汞循环代谢：

- **脱甲基反应**：CH₃Hg⁺ → Hg²⁺ + CH₄（甲烷）
- **还原反应**：Hg²⁺ → Hg⁰（元素汞，挥发性）
- **净效果**：肠道中的无机汞被细菌转化为**元素汞蒸气（Hg⁰）**——这是一种在室温下即可挥发的银白色金属蒸气

**叠加路径——伴生气体**：

同样的肠道细菌在分解未消化的蛋白质和纤维素时，还会产生：

| 气体 | 化学式 | 气味 | 浓度效应 |
|------|--------|------|---------|
| **甲硫醇** | CH₃SH | 低浓度时甜味、卷心菜味 | 微量即明显 |
| **硫化氢** | H₂S | 低浓度时有甜味，高浓度臭鸡蛋 | 低浓度可被感知 |
| **二甲基硫醚** | (CH₃)₂S | 卷心菜/海腥味 | 与甲硫醇协同增强气味 |
| **甲烷** | CH₄ | 无味 | 大量产生（占比最大） |
| **水蒸气** | H₂O | — | 冷凝后形成可见白色雾气 |

**关键结论**：可见的白色"毒气"其实是**水蒸气冷凝（雾）+ 伴生有机硫化物的复合体**。真正的杀死力（Hg⁰蒸气）是不可见且无味的。那甜甜的、"刺鼻"的味道不是毒气本身的气味，是**尸气**——一个有用的预警信号。

#### 为什么Hg⁰蒸气是即死级的

元素汞蒸气（Hg⁰）吸入后的毒性路径与甲基汞（CH₃Hg⁺）不同：

| 特性 | 甲基汞（水媒） | 汞蒸气（空气媒） |
|------|--------------|-----------------|
| **进入途径** | 消化道 → 缓慢入血 | 肺部 → 80%一次吸入即入血 |
| **血脑屏障** | 主动转运，高效穿透 | 被动扩散，穿透效率低于甲基汞 |
| **靶器官** | 中枢神经（小脑/皮层） | 肺 + 中枢神经 |
| **潜伏期** | 6~72小时 | **数分钟**即开始出现症状 |
| **症状链** | 口渴→共济失调→攻击性 | 头晕→恶心→视野模糊→**快速意识丧失** |

**核心差异**：Hg⁰蒸气吸入后，在肺泡中迅速氧化为Hg²⁺并滞留，同时部分未经氧化的Hg⁰直接通过血脑屏障进入脑组织，在脑细胞内被氧化为Hg²⁺后**无法再排出**，形成不可逆的脑损伤。这个过程在**数分钟**内就能达到足以致死的浓度——远比喝污染水的累积速度快。

> **游戏设计推论**：为什么进风机房的场景中，闻到气味后"头晕目眩"然后快速意识模糊？因为Hg⁰蒸气的急性中毒窗口只有**几分钟**。防毒面具中的活性炭层对汞蒸气有良好的吸附能力，所以是有效对策。

#### 为什么丧尸自己不被毒死

丧尸已经处于"甲基汞完全摧毁高级神经"的状态：
- 大脑皮层、小脑、基底节——全部已损坏
- 剩下的只有脑干反射（呼吸、心跳、本能的行走和咬合）
- Hg⁰蒸气进入丧尸体内后，同样会氧化为Hg²⁺并在脑组织中沉积
- 但**已经没有"可以被进一步破坏的功能结构"了**
- 结论：毒气丧尸不是"免疫"，是"已经没什么可摧毁的了"

**比喻**：你在一栋已经烧成空壳的建筑里放一把火——火会烧得更旺（因为汞蒸气在丧尸体内的沉积会使它的组织汞浓度更高），但对建筑本身来说，它已经没什么可烧的了。

### 4.3 物理特征——为什么毒气丧尸看起来"魁梧"

目前游戏文本中描述毒气丧尸为"魁梧的"（初遇毒气型丧尸）和"一只穿着深蓝色工装裤的手从柜台下面伸了出来"（五金店）。这个体型特征有生理基础：

**三个因素叠加导致的魁梧体型**：

1. **腹部膨隆（最主要）**：肠道中持续产生的气体（甲烷+Hg⁰+伴生气体）使腹腔显著膨胀。在死后24-48小时内，这个膨胀过程达到峰值。衣物（尤其是工装裤/工作服）被绷紧，形成一个"将军肚"的轮廓。

2. **皮下气肿**：在少数严重案例中，肠道气压过高导致气体渗入肠壁→进入腹腔→沿筋膜层扩散到皮下组织。最明显的部位是：胸壁、颈部、锁骨上窝。这些部位的皮肤被气体撑起，使上半身看起来异常宽厚。

3. **组织水肿**：汞中毒导致的肾功能衰竭使大量水分滞留体内，进一步增加总体积。

**与普通丧尸的体型差异**：

| 部位 | 普通丧尸 | 毒气丧尸 |
|------|---------|---------|
| 腹部 | 平坦或微凹（脱水） | **明显隆起**，绷紧 |
| 胸廓 | 正常 | 略呈桶状（皮下气肿） |
| 颈部 | 正常 | 增粗（皮下气肿最明显处） |
| 步态 | 拖沓、重心不稳 | **双腿微开**（为腹部腾空间），重心略微后倾 |
| 衣服 | 松垮（脱水后变瘦） | **紧绷**，尤其腰腹和领口 |

**不是所有腹部膨隆的丧尸都是毒气型的**——部分普通丧尸在死后也会因肠道产气而轻度膨胀。毒气丧尸的核心鉴别点是：**静止时腹部有明显蠕动**（肠道的产气细菌仍在活动）和**周围有可见的白色雾霾**（水蒸气冷凝）。

### 4.4 行为特征

#### 趋密闭性（区别于普通丧尸的趋水性）

毒气丧尸的行为规律由其生理状态决定：

- **主动寻找密闭空间**：这不是意识行为，是一种"反射性趋向"——密闭空间内自身的Hg⁰蒸气浓度更高，湿度更高，产气细菌在潮湿环境中活性更强→身体感知到"舒适"信号→向密闭空间移动
- **与普通丧尸的关系**：毒气丧尸对普通丧尸有**被动吸引效应**：
  1. 毒气丧尸进入一个密闭房间
  2. 它呼出的水蒸气使房间湿度升高
  3. 趋水性的普通丧尸被湿度吸引，朝房间聚集
  4. 结果：**毒气丧尸周围会自然形成丧尸簇**
- **通风驱散**：如果密闭空间被通风（打开门、启动风机），毒气丧尸会表现出明显的烦躁和不安——因为"舒适"的微环境被破坏了

**与现有游戏内容的对应**：

- **民防设施**：地下民防设施是一个天然密闭环境，通风不良 → 毒气丧尸的理想栖息地。进风机房可以启动风机驱散气体——游戏里已有"打开鼓风机"的日记提示
- **五金店**：五金店室内空间，没有通风 → 气体积累。玩家从侧窗进入时相对安全（靠近室外），绕到工具区内部时开始出现症状

#### "绿色眼睛"的解释

游戏文本中描述毒气丧尸的眼睛为"绿色的亮斑""绿色的眼睛"（进风机房失败结局、死路一条结局）。

**科学解释**：
- 正常人的视网膜在死亡后会迅速变混浊（角膜混浊、玻璃体混浊）
- 汞中毒的丧尸，因为汞的防腐作用，眼球组织的腐败速度**显著慢于正常尸体**
- 但汞沉积在视网膜色素上皮层（RPE）中，被汞离子染色的RPE细胞在特定光照下呈现**暗绿色荧光**
- 当光线从侧面照射到丧尸的眼睛时，被染色的RPE层反射出绿色光泽——类似猫科动物的**tapetum lucidum（明毯）**效应，但机制不同
- 在完全黑暗的环境中，这个效应最明显——玩家在黑暗中看到两点绿光，不是因为丧尸的眼睛会发光，是环境中有微弱的背景光（月光、应急灯）被汞沉积的视网膜反射回来

> **游戏设计推论**：这就是为什么进风机房失败结局和死路一条结局中都提到了"绿色的眼睛"——这两个场景都是黑暗环境。在其他场景（有光的环境）中，毒气丧尸的眼睛看起来只是暗沉无光，不会发出那种诡异的绿光。这也暗示：**如果你在黑暗中看到了绿光，你面对的很可能是毒气丧尸。**

### 4.5 毒气丧尸在游戏中的统一设计

#### 出场条件

- **环境**：位于**密闭或半密闭**的室内空间，通风不良
- **本体外观**：
  - 腹部明显隆起（衣服绷紧）
  - 周围有淡淡的白色雾气（尤其在温度较低的环境中明显）
  - 黑暗中可见绿色荧光眼睛
- **预告信号**：玩家进入该区域前50m左右，可以闻到一股**淡淡的甜味**（甲硫醇的低浓度气味）+ 隐约的湿气感

#### 战斗机制

| 情况 | 反应 | 结果 |
|------|------|------|
| 玩家进入密闭空间，未戴防毒面具 | → 快速积累Hg⁰蒸气吸入 | 数分钟内头晕→意识模糊→死亡 |
| 玩家戴防毒面具进入 | → Hg⁰蒸气被活性炭吸附 | 安全，可正常互动 |
| 玩家**未戴**面具但**打开了通风（门/窗/风机）** | → 气体稀释，但需要时间 | 安全窗口延长，但仍需尽快离开 |
| 玩家在开放空间遭遇毒气丧尸 | → 气体随风消散 | 基本无害——只比普通丧尸多一个"腹部膨隆"的特征 |

**如果玩家在毒气环境中被击倒（非死亡）**：
- 汞蒸气暴露后不一定立即致死
- 幸存后 long-term effect：mercuryLoad +10~20（如果使用汞沉积量系统）
- 症状：持续头痛、手指震颤、视野中出现盲点——可以用于后续剧情伏笔

#### 击杀后环境

毒气丧尸被击杀后，其腹部继续产气约6-12小时（肠道细菌仍在活动），然后停止。房间内的气体浓度在击杀后约**30分钟**通过自然通风降至安全水平。如果房间无通风，气体可持续存在**数天**。

### 4.6 对现有场景的重新梳理

**① 民防设施-初遇毒气型丧尸**

**② 民防设施-进风机房-失败结局**

**③ 五金店-毒气结局**

现状：气体来源是"柜台下方有一根管道裂了"——这属于环境灾害，和丧尸无关。丧尸只是"穿着深蓝色工装裤的丧尸躲在毒气里"。

建议：保留"管道泄漏"的设定，这是独立的环境灾害。但可以改为：**这根管道泄漏的气体吸引了毒气丧尸**（密闭+湿润环境），所以毒气丧尸不是"碰巧在这里"，而是**专门在这里**。玩家在前面的探索中如果能识别出毒气丧尸的预告信号（甜味、雾气），就能提前预警。

#### 统一的叙事逻辑

```
毒气丧尸诞生的完整链条：

生前职业暴露（芜湖工人/化工厂周边居民/长期饮用污染水）
      ↓
肠道菌群适应高汞环境 → 产汞菌群比例升高
      ↓
感染/死亡后，肠道菌群持续活动
      ↓
产生Hg⁰蒸气 + 伴生气体 → 腹部膨隆 + 白色雾气
      ↓
趋密闭性 → 自动寻找地下室/暗室/通风不良的房间
      ↓
呼出湿气 + 气体积累 → 房间湿度上升
      ↓
趋水性的普通丧尸被吸引 → 毒气丧尸周围自然形成丧尸簇

结果：一个密闭房间 = 一个天然陷阱
```

**进风机房作为这个陷阱的破局点**：

## 五、"黑皮"与注射器的科学解释

### 5.1 现有设定

在益丰大药房的库房中，玩家遭遇一只特征鲜明的丧尸：
- **外观**："它的身体被覆盖了一层黑色的皮"（黑皮）
- **手持**：攥着一个针筒
- **行为**：主动攻击，需通过记忆闪色QTE击败
- **位置**：药房库房，位于利昂（前台）和赵广成（办公室）之间的过渡区域

需要解释的核心问题：
1. 为什么这只丧尸的皮肤是黑色的——和其他丧尸不一样？
2. 为什么它手里拿着针筒？针筒里有什么？

### 5.2 "黑皮"的机制——汞负荷过高导致的皮肤黑变（Mercury-Induced Melanoderma）

#### 核心原理

黑皮不是一种"变异"，而是**汞负荷超过临界值的直观外在标志**。

正常丧尸的皮肤颜色演变（来自一、）：
```
面部潮红（潜伏期）→ 灰白/蜡黄（进展期）→ 青紫（重症期）→ 暗紫/褐绿（终末期）
```

"黑皮"丧尸的皮肤走了一条不同的路径：**不是因为组织缺氧或腐败变黑，而是汞在皮肤中的化学沉积导致的色素改变。**

#### 分步机制

**第一步——生前：慢性汞暴露引起的黑素细胞活化**

甲基汞对黑素细胞（melanocytes）有特殊的刺激作用：
- 汞离子（Hg²⁺）与酪氨酸酶（tyrosinase）结合，**不可逆地激活该酶**
- 酪氨酸酶是黑色素合成的关键酶 → 被永久激活 → 黑色素大量生成
- 同时，汞离子与黑素体中的黑色素分子结合，形成**深灰色的汞-黑色素络合物**
- 这种络合物的颜色比普通黑色素更暗、更偏灰蓝色

**证据**：在职业性汞暴露人群中，"汞性黑变病（mercurial melanoderma）"是有文献记载的症状。患者的皮肤（尤其是面部、颈部、手背）出现弥漫性的灰蓝色至暗灰色色素沉着。**这不是极端的假设，而是已知的汞毒理学现象。**

**第二步——死后：汞的防腐作用改变了腐败颜色**

正常情况下，尸体腐败会使皮肤呈绿黑色（硫化血红蛋白 + 细菌色素）。但汞的抗菌作用抑制了腐败菌的生长，使皮肤没有经历"绿黑→褐绿"的典型腐败变色。

取而代之的是：
- 皮肤中的汞-黑色素络合物在组织脱水过程中**浓缩**
- 皮下微血管中的血液逐渐氧化 → 血红蛋白中的铁与汞结合，形成**黑色的硫化汞（HgS，即朱砂的黑色同素异形体）**的痕迹
- 皮肤最外层（角质层）缓慢干化，形成一层**黑色的、类似羊皮纸的薄层**

**文字效果**："覆盖了一层黑色的皮"——玩家的视觉描述非常准确。这不是皮肤"变成了"黑色，而是皮肤表面形成了**一层干化、暗色、类似角质层的覆盖物**。

#### 为什么只有这只丧尸是黑皮的

需要**同时满足**两个条件：
1. **生前已有慢性汞暴露**（黑素细胞被激活，黑色素-汞络合物已经形成）
2. **死后未经历正常腐败**（汞的防腐作用使皮肤得以保留并干化成黑色）

绝大多数丧尸只满足条件2（死后未腐败），但不满足条件1（生前没有长期汞暴露）。所以它们的肤色演变走的是"灰白→青紫→暗紫"路径，而不是"变黑"路径。

**药房库房丧尸的身份**：此人正是药房的小伙子学徒——**李明浩**（石蜡油标签上的签名者）。他在爆发前几天就因饮用污染水而中毒（人物档案："利昂是被小伙子学徒咬伤的——这个小伙子前几天就因为喝了被污染的水中毒"），是药房内最早达到高汞负荷的人。他的黑皮是**慢性暴露 + 急性爆发叠加**的结果：前几天的持续饮水使基线汞负荷远超常人，比普通丧尸多出了数天的汞积累时间，黑素细胞被充分激活。

> **游戏意义**：黑皮是"汞负荷超标"的视觉标志。如果玩家在游戏过程中注意到自己角色的皮肤颜色变化（通过镜子、水面倒影等），可以作为一个隐藏的预警——玩家自己的mercuryLoad正在攀升。

### 5.3 注射器的解释

#### 针筒里有什么

玩家击败丧尸后可以拿到针筒。设定如下：

**针筒是空的。曾经装着水。**

场景（现已实装于游戏中）：
1. 李明浩在发病初期出现了严重的口渴症状（甲基汞中毒的早期标志）
2. 他在库房找水喝，极度口渴下用医用注射器从水龙头或桶装水中抽水
3. 来不及喝完，他发病了。针筒还攥在手里
4. 针头被他用手硬掰断了——可能是想加快抽水的速度，也可能是手已经开始抖了

**为什么用针筒喝水**：
- 库房里没有杯子，只有医用器具
- 极度口渴的冲动下，人会使用任何能找到的容器
- 现实中汞中毒的极度口渴是"大脑层面的求生式渴感"，行为会变得不理性

> **讽刺之处**：他前几天就喝了污染水→中毒→口渴→到库房找水→用针筒喝的水仍然是同一套供水系统的污染水→加速汞积累→最终在黑皮状态下变成丧尸。他是药房里最早中毒的人，也是最彻底的那个。

**与利昂之死的关联**：
- 李明浩发病后攻击了利昂（咬伤），利昂将他锁在库房中
- 利昂自己被咬伤，在前台度过了最后的清醒时间
- 赵广成到店发现出事，躲进办公室
- 林小满（长发女学徒）可能在另一条走廊上被困
- 药房的四条命运线在同一个下午完成

#### 为什么针筒丧尸比普通丧尸更具攻击性

游戏中的战斗表现（记忆闪色QTE）暗示这只丧尸更难对付。理由：
- **更高的汞负荷** = 更彻底的抑制解除 = 更强的攻击驱动力
- **痛觉消失更彻底**（普通丧尸可能还残留微量痛觉，这只完全没有）
- **肌肉力量更大**（100%发力已经持续了更长时间）
- 外壳描述："它看起来像一个普通的人，但是它的身体被覆盖了一层黑色的皮，散发着一股死亡的气味"——这层黑皮（干化角质层 + 汞-黑色素络合物）在物理上也更硬，可能使普通打击效果打折扣

### 5.4 对游戏场景的微调建议

现状文本已有：玩家看到它手上的针筒，击败后继续去办公室找赵广成。

补充建议（可选，不影响现有剧情链）：
- 击败后，可增加一个"检查针筒"的选项（或不增加，让针筒的细节作为一种视觉叙事存在）
- 如果检查：发现针筒内壁有少量水渍、针头被折断丢弃在旁边的垃圾桶里
- 这个发现不与任何主线任务绑定——纯环境叙事，留给细心的玩家作为"世界信息"的拼图碎片

**不推荐做成任务道具**。针筒就是一个普通人的最后时刻的物件，和利昂的收据、林小满的胸牌一样，属于"环境叙事"层的信息。不触发任何任务、不解锁任何技能、不打开任何新区域——但它让玩家停下来想一下："这个人死前到底经历了什么？"

## 六、丧尸的尸体分解与气味系统

### 6.1 核心前提：被咬死的丧尸 = 一具普通尸体

甲基汞中毒者本质上还是**活人**，不是亡灵。死了就是死了——没有二次尸变，没有"病毒继续繁殖"。剩下的就是**正常的生物分解过程**。

唯一特殊之处：**生前甲基汞负荷高的个体，汞的抑菌作用会延缓腐败速度**（和"黑皮"丧尸同理）。但这是延缓，不是停止。

### 6.2 分解时间表（6月底~7月初上海，30℃+，高湿度）

| 时间 | 状态 | 气味 | 昆虫活动 |
|------|------|------|---------|
| **死后0~6小时** | 尸斑开始形成，尸僵尚未出现 | 几乎无味 | 苍蝇开始产卵于口鼻、伤口、眼睑 |
| **6~12小时** | 尸僵达峰值，皮肤开始变色（灰白→青紫） | 极轻微，需贴近才能闻到 | 蝇卵孵化中 |
| **12~24小时** | 肠道菌群开始工作，腹部膨胀 | 低浓度腐胺/尸胺，有"不好闻"的感觉但尚可忍受 | 蛆开始孵化，集中在伤口和体腔开口 |
| **24~48小时** | 腹部明显膨胀，皮肤出现腐败静脉网 | **明显腐臭**——甜腻的腐败味开始扩散，在密闭空间中累积到让人皱眉的程度 | 蛆大量活动，加速软组织分解 |
| **48~72小时** | 组织液渗出，尸体整体软化 | **浓烈尸臭**——甜腻+氨味+硫化氢，露天环境下数米可闻，密闭空间内"推门直接被熏吐" | 蛆进入暴食期，体腔高度充满蛆群 |
| **3~7天** | 软组织大幅分解，骨架开始暴露 | 恶臭达到峰值后逐渐转为"陈尸臭"（淡但持久，顽固附着于衣物和墙面） | 蛆成熟化蛹→成蝇离开；甲虫类开始参与 |
| **7天+** | 白骨化（露天快/室内慢） | 基本无味 | 仅剩耐旱节肢动物 |

**关键游戏化提炼**：
- Day 1~2（6/29~6/30）：尸体大多在死后24~48h内，**有明显臭味但尚可接近**。密闭空间开始累积高浓度气味。
- Day 3~5（7/1~7/3）：**恶臭期**。室内尸体基本不可接近——如果某个房间有重要物资但横着一具三天以上的尸体，玩家需要决定值不值得。
- Day 7+：露天尸体白骨化或半白骨化，气味消散。但**尸体曾经躺过的位置会留下永久性的污渍和气味残留**（玩家回到一个曾经有大量尸体的场景时会感觉到"这里还是那么臭"——哪怕尸体已经搬走了）。

### 6.3 活着的丧尸有什么味道

与腐烂的尸体不同，活着的丧尸（感染者）身上有一种**特征性气味**——来自甲基汞代谢产物：

**肠源性汞挥发综合征的附带产物**：肠道细菌分解含硫氨基酸产生的甲硫醇、二甲基硫醚等有机硫化物，在低浓度下产生一种**甜腻的、带金属感的卷心菜味**。很像劣质口香糖和铜钱混在一起的味道。

- **不是腐臭味**，是一种更"奇怪"的气味——大多数人闻到后会本能地觉得"不对劲"，但不一定能说出是什么
- **密闭空间中的丧尸群**会让这种气味积累到明显可辨的程度
- **毒气丧尸**周围这种气味最浓（因为产气量最大），还叠加了水蒸气冷凝的"湿感"

**游戏设计推论**：

1. **早期伏笔**：玩家在Day 1经过某个密闭空间（如民防设施通道、老洪的房间门缝）时闻到一种"说不清的甜腻味"，但当时没有线索解释——直到后来找到王知筠的资料才知道那是甲基汞代谢产物的气味。"原来是这个味道"——一个跨越多天的认知串联。

2. **预警信号**：甜腻味越浓 → 前方丧尸密度越高（或有一只毒气丧尸）。玩家学会这个规律后，可以作为一种"嗅觉雷达"来辅助决策。

3. **老洪的房间**：他绑在椅子上绑了两天，房间里的气味应该是**甜腻味（活着的感染者）+ 逐渐开始的腐臭味（组织坏死/伤口感染）**的混合——老洪的房间里应该是整个三林路区域"最难闻"的地方之一。

### 6.4 尸体堆积——哪些地方尸体最多

尸体的分布不是均匀的。以下因素决定了尸体的堆积密度：

**核心逻辑**：**人群密度 × 封锁/逃生失败率 = 尸体堆积量**

#### S级（堆积成山）——尸体会叠在一起的地方

| 地点 | 堆积原因 | 尸体数量估计 | 游戏建议 |
|------|---------|-------------|---------|
| **地铁站出入口** | 6/28政府封锁令下达后，大量市民涌向地铁试图离开→站内已有感染者→出入口人群双向对冲→踩踏+感染连锁反应。**楼梯底部的人被上面倒下的人堆埋** | 单站出入口数十至上百具 | 11号线三林东路站和6号线华夏西路站的出入口楼梯底部应出现"人堆"——玩家的脚会踩到什么软的东西 |
| **高架/高速匝道口** | 6/28晚大量私家车涌上高架→堵死→车内人下车→在高架路面上变成丧尸→后续车辆持续拥堵→最终形成一条**数公里长的"废弃车辆+尸体"混合带**。匝道汇入处是最严重的堆点 | 匝道口数十具，全线散布 | 杨高南路高架上桥口：车的缝隙中偶尔能看到倒地的尸体，有些被车辆碾过。玩家在高架上行走需要从尸体和车辆的间隙中穿行 |
| **大型商场玻璃门/旋转门** | 人群试图逃离商场→玻璃门一次只能通过有限人数→后面的人推前面的人→前面的人被压在门上/倒在地上一堆人被踩过去。金谊广场和新达汇的正门都是高风险点 | 单门数具至十余具 | 可以设计一个场景：旋转门的格子里卡着尸体，玻璃上全是血手印 |

|------|---------|---------|
| **医院急诊室/大门口** | 6/28第一批伤者被送到医院→医院爆发→急诊室最先沦陷。送来的伤者+来接人的家属+试图逃出的医护人员在门口形成尸体簇。**仁济南院是王知筠的死亡地点**——其急诊通道的尸密度应该在全游戏排前三 | 仁济南院如果作为可探索场景，门口草坪和救护车通道上横七竖八倒着穿病号服和穿白大褂的尸体——"这里曾经有人试图救援，但他们自己也倒下了" |

| **公交车站** | 等车的人 |

| 地点 | 说明 |
|------|------|

| **地下车库** | 末日初期有人躲在地下车库的车里→车外丧尸路过→车内人不敢出来→渴死/饿死/热死。地库通风差，尸臭极其持久 |

### 6.5 气味作为游戏机制

#### 气味分层表（从入口到接近尸体）

| 距离 | 气味描述 | 可识别的提示 |
|------|---------|-------------|
| 50m外（室外） | 无异常 | — |
| 20~30m（室外） | 隐约的"不对劲"——说不出是什么，但风向转过来了就闻到了 | 经验丰富的玩家会警觉 |
| 5~10m | **甜腻味（活尸）或腐臭味（尸体）**明确可辨 | 可判断前方是活尸群还是陈尸点 |
| 1~3m | 浓烈到让人本能后仰 | 室内场景此时开始触发"反胃"反应 |
| 接触到尸体 | 气味粘在衣服上——离开后还能闻到自己身上的味道 | 适合在文本中写一句"你离开后还在衣领上闻到那股味道" |

#### 设计的两个原则

1. **气味不是"系统提示"**——它是环境叙事的一部分，通过文本描述传递给玩家。不要在界面上显示"你闻到一股味道"的状态条，而是在场景文字中自然带出。

2. **前期少写，后期多写**——Day 1玩家对气味的感知应该是模糊的（"总觉得空气不太对"），Day 5之后可以直接写"浓烈的尸臭"。这个变化暗含了两层意思：尸体确实在腐烂，以及玩家在逐渐习惯这个味道。

### 6.6 蛆在游戏中的使用

蛆是尸体分解的自然产物，在合适的时机出现在合适的场景中可以增强真实感。但它的"杀伤力"主要是视觉/心理层面的，不适合滥用。

**建议使用的场景**：

- **6/30之后任何室内尸体**：稍微掀开衣服就能看到蛆在蠕动——用一笔带过的方式写（"掀开衣服的时候你看到……你没有再看第二眼"），玩家脑补的效果最好
- **大规模堆积的尸体**（地铁出入口、高架匝道）：不用近距离看到蛆，但远处苍蝇的嗡嗡声可以作为预警——**苍蝇密度和尸体数量成正比**
- **不建议**：把蛆作为解密元素或交互道具。它应该始终是环境描写的点缀，不是玩法的核心。

**一个具体的叙事可能性**：

> 玩家在Day 5左右回到某个六天前经过的地点——当时那里有一具尸体，玩家没有管它。
> 六天后尸体还在，但已经完全变了样：膨胀、变色、苍蝇环绕。玩家可能会想"我上次经过的时候它还是个人样"——这种对比本身就在叙事时间的流逝。

- 退休化工厂工人**洪国栋（老洪）**当天下午与上海某高校环境毒理学方向的青年教师**王知筠**通了电话。他在电话里说了一句话——_"不是病毒，是汞。你们搞这个的应该懂。"_
- 王知筠此前已经从自己的实验数据中察觉异常（6/24黄浦江采样数据跳变），老洪的电话把她最后的怀疑变成了确信。

- 她留下的**手机**里有一段没发出去的B站视频草稿和实验记录本的扫描件——**这是游戏里可找到的关键证据之一**，直接指向"不是病毒，是甲基汞"的真相。

## 八、玩家昏迷设定——"为什么6/29才醒来"

### 时间线

| 时间 | 事件 |
|------|------|
| **6/27 晚** | 玩家（樱桃苑某栋2楼，刚高考完的毕业生）在家，喝了自来水 |
| **6/28 凌晨~上午** | 玩家开始发烧（轻度甲基汞中毒的免疫反应），39°C+，意识模糊 |
| **6/28 上午~中午** | 母亲发现玩家高烧不退，打120/110均占线，打不到车 |
| **6/28 下午** | 母亲决定出门去最近的医院（东方医院南院/三林社区卫生服务中心）找人帮忙。走前在玩家额头放了湿毛巾，床头柜留了字条 |
| **6/28 傍晚~深夜** | 母亲在路上遭遇失控者，被卷入混乱。**她没能回来** |
| **6/28 深夜~6/29 凌晨** | 玩家持续高烧昏迷。因摄入量不高，身体自行代谢微量甲基汞，未突破重症阈值 |
| **6/29 7:30~8:00** | 玩家退烧，醒来。闹钟停了。母亲不在家。**游戏开始** |

### 为什么玩家没变成丧尸

- 玩家在昏迷中完成了自身的微量甲基汞代谢，血汞回落到亚临床水平
- 退烧后除了轻微口渴和头晕，**没有变成丧尸**
- 为后续"半感染"线留了空间：如果被咬伤且持续饮用污染水，血汞可能重新攀升，进入"半丧尸化"状态

### 设计意图

1. **解释玩家为什么错过了6/28的暴乱**：发烧昏迷了一整天。"起床→开门→丧尸糊脸"是常见初见杀——正常人在6/28就知道不该出门了，但玩家刚刚醒。

2. **母亲的去向作为情感锚点**：
   - 玩家提到的"妈妈买早餐还没回来"是**温情但错误**的记忆——母亲不是去买早餐了，是去找人救他
   - 游戏中可设一条**隐线**：字条→邻居目击→小区监控（如果能通电）→街道目击→医院线索
   - 她的具体命运留给玩家探索——不设死，只设线索

3. **玩家的低血汞**为后续"半感染"线留了空间。

### 游戏中已有的伏笔和需新增内容

**已有伏笔**：
- 闹钟停在7:30——玩家以为是"刚过7:30"，实际昨晚就没再走过
- "再过半小时妈妈就买早餐回来了"——玩家对时间的感知是错误的
- 多次回卧室后闹钟依然显示7:30——暗示钟坏了

**需新增的初期线索**：
- **床头柜**：母亲留下的字条——_"妈去找人帮忙，你躺着别动。"_
- **湿毛巾**：还搭在额头上，已经干透了
- **母亲的卧室**：手机留在充电器上，包不在——她出门时带走了钥匙和零钱

**需调整的文本**：
- 玩家6/29上午说的"再过半小时妈妈就买早餐回来了" → 建议改为更模糊的 _"妈妈怎么还没回来"_
- 玄关"是妈妈还是查水表的"→ 保留，但玩家看到字条后这个选项的意味会完全不同

## 十二、备注

1. **这份时间线是基线，不是枷锁**：具体场景开发可在基础上微调，但核心锚点（6/28爆发、6/29玩家醒来、7/2陈默被偷、7月中旬秩序成形）应保持一致。
2. **王知筠的手机**是连接科学真相和玩家认知的关键桥梁，建议在游戏中期（如进入仁济南院或某个医院场景）让玩家找到。实验记录本可放在老洪的房间里与之对应——两人通过电话交换了信息，但直到死后他们的线索才被玩家拼在一起。（角色名可调整）
3. **中央专家组被困在长三角外围**有两个用途：解释"救援不会来了"（不是政府不作为，是进不来）；为后期"向外突围"路线提供叙事出口（如从崇明岛坐船去南通与外围部队汇合）。
4. **舆论管控**不需要在游戏中大篇幅展现，但可通过捡到的手机里的时间线碎片建立暗示——玩家会发现自己"没看到任何预警信息"这件事本身就很可疑。
5. **玩家母亲的命运**设计时注意不要设定得太死。她的下落是一条可选追踪的线索，不是主线必经之路。有的玩家会去找，有的不会；找到了可能有好的结果，也可能没有——但无论哪种，都不应该直接通向"这是真相"的核心判断。

# 第三部分：尸潮扩散模型

> 从水道、铁路、公路三张网络推演甲基汞污染如何从芜湖一个点扩散为覆盖长三角的尸潮大灾变。

## 十三、长三角尸潮扩散模型

本文档将从**地面真相**出发，沿水道、铁路、公路三张网络，推演甲基汞污染如何从芜湖融汇化工一个点，扩散为覆盖整个长三角的尸潮大灾变。

> **阅读提示**：以下内容为世界观基线，不意味着玩家需要在游戏中走遍所有地方。但每条路线每个区域的尸潮状态，会影响广播消息、幸存者传闻、遭遇战难度等具体设计。

---

### 13.1 底层模型：两波传播论

丧尸的扩散由两个不同速度的波叠加构成：

| 波次 | 媒介 | 速度 | 覆盖范围 |
|------|------|------|---------|
| **第一波 · 水媒** | 饮用水（自来水/桶装水/地表水） | 慢，按水系流动速度（约3~5km/h水流） | **长三角全境**——所有人都可能喝到污染水 |
| **第二波 · 体液** | 咬伤/抓伤的唾液接触（人际传播） | 快，按感染者活动速度（约3~5km/h步行） | 局部爆发后以爆发点为中心辐射 |

**关键叠加效应**：第一波决定**"哪里会爆发"**，第二波决定**"爆发后扩散多快"**。当两波重叠（即一个城市同时有大量水媒中毒者和被咬伤者），爆发曲线呈指数上升。

### 13.2 污染羽流与长三角水网

#### 污染源的长江之旅

从芜湖融汇化工渗出的无机汞进入长江后，沿江向下游输送。这是一个**缓慢推进的污染羽流**：

```
芜湖 → 马鞍山 → 南京 → 镇江 → 扬州/泰州(北岸) → 常州/无锡/苏州(南岸) → 南通 → 上海
|---- 约 50km ----|-- 约 70km --|---- 约 120km ----|---- 约 150km ----|-- 约 60km --|
```

但水媒传播**不等于**每个城市在同一时间爆发。关键变量是**自来水厂的取水口位置和供水范围**：

| 城市 | 主要水源 | 污染到达时间 | 关键变量 |
|------|---------|-------------|---------|
| **芜湖** | 长江（紧邻污染源） | **6/20~6/22**（浓度最高） | 最早。部分老城区供水管与废弃工厂管网有交叉 |
| **马鞍山** | 长江 | 6/22~6/24 | 钢铁工业用水占比大，居民用水受影响略晚 |
| **南京** | 长江（夹江/燕子矶等取水口） | 6/23~6/25 | 江北/江南多个水厂，受污染程度不均 |
| **镇江/扬州** | 长江 | 6/24~6/26 | 运河枢纽，南北扩散的分叉点 |
| **常州/无锡** | 长江（部分）+ 太湖 | 6/24~6/27 | 依赖太湖供水的区域爆发更晚（太湖自净稀释） |
| **苏州** | 太湖 + 长江 | 6/25~6/28 | 太湖东岸取水，污染衰减但积累 |
| **南通** | 长江（南通段） | 6/25~6/28 | 北岸，水流稍慢 |
| **上海** | 黄浦江上游 + 陈行水库/青草沙 | **6/25~6/28** | 距离最远但人口最多，稀释后的总暴露量仍然巨大 |

> **关键结论**：不是"芜湖先爆，然后依次往下"。因为潜伏期（6-72h）的存在，**6/25~6/28之间长江沿线主要城市几乎是同时出现首发症状的**——芜湖的浓度最高所以爆发最剧烈，但下游城市因为人口基数大，绝对数量不低。

#### 太湖流域——第二条扩散轴线

太湖承接了来自长江的补给水（通过望虞河、太浦河等），也接纳了来自上游的汞污染。太湖流域的水网密度极高——苏州、无锡、湖州环绕太湖，城市之间河道纵横。

```
                 长江（主污染带）
                   |  ↓
       望虞河 ←—— 闸 → 太湖
                   |      ↓
       无锡 ←—— 梁溪河 → 太湖
       苏州 ←—— 胥江/娄江 → 太湖
       湖州 ←—— 苕溪 → 太湖
```

太湖流域的爆发特点是：**不是点状爆发，而是面状爆发**。水网将污染均匀散布到整个流域，各城镇几乎同时进入警戒状态。

#### 第四扩散轴线：京杭大运河

大运河起自北京，经山东、江苏至杭州，在长江以南段（镇江—苏州—嘉兴—杭州）与自然水系高度重合。运河水流极慢（几乎静止），但运河沿岸的古镇和居民区密集。

大运河在江南的主要路径：
```
镇江 → 丹阳 → 常州 → 无锡 → 苏州 → 吴江 → 嘉兴 → 杭州
```

大运河的意义：**它是一条穿越城市核心区的连续水道**。丧尸沿运河走，不需要经过任何开放地形，就能从镇江一路走到杭州。这意味着江南运河沿线在爆发后一周内会形成一条**连续的高密度丧尸走廊**，将所有沿线城市"缝合"成一个超大死亡区。

### 13.3 三张地下网络：尸潮如何跨越物理阻隔

丧尸的地面移动受建筑、围墙、河流阻挡。但**地下通道是畅通的**。长三角有三层地下网络决定了尸潮的跨区域扩散速度。

#### 第一层：地铁隧道（最快、最广）

截至2026年，长三角地铁通车里程超过2000km，是全球最大的都市地铁群：

| 城市 | 运营里程 | 重要连接线 |
|------|---------|-----------|
| **上海** | ~900km（19条） | 11号线→苏州S1线（全球最长的跨省地铁） |
| **南京** | ~500km（14条） | S6号线通句容、S9号线通高淳 |
| **杭州** | ~500km（16条） | 杭海城际通海宁杭绍城际通绍兴 |
| **苏州** | ~300km（9条） | S1线通上海11号线 |
| **无锡** | ~130km（5条） | 与苏州接壤的城际规划未完全开通 |
| **宁波** | ~200km（6条） | — |

**扩散场景推演：**

6/28 下午~晚间，各地铁站陆续出现感染者。地铁公司在部分线路停运前，已经有列车载着感染者穿越整条线路。

```
6/28 14:00    上海11号线某列车在御桥站出现感染者
6/28 15:30    列车抵达罗山路站，感染者换乘16号线往滴水湖方向
6/28 16:00    另一方向列车抵达花桥站（苏州界），感染者进入苏州S1线
6/28 17:00    苏州S1线感染者进入苏州地铁网络，在广济南路站换乘1号线/2号线
6/28 18:00    苏州1号线/2号线上的感染者到达苏州全城各站
6/29 凌晨     上海、苏州、南京、杭州的地下隧道中已有大量丧尸在持续移动
```

> **关键结论**：地铁系统让丧尸的跨区域扩散从**天**压缩到**小时**。苏州和上海的"同步沦陷"不是因为水污染的时间巧合，而是因为11号线在物理上连接了两座城市——即使苏州的水污染到达时间比上海晚6-12小时，丧尸也会通过地铁提前到达。

#### 第二层：排水暗渠/综合管廊（持续性）

城市的地下排水系统比地铁网络更密集、更隐蔽。雨水管、污水管、合流管的直径从30cm到3m不等。长三角多数城市的地下管廊和排水主干渠**人在其中可以直立行走**。

扩散特点：
- **无方向性**：排水暗渠没有"线路图"，丧尸在其中的扩散更像渗流——沿管径大的主管渠优先走
- **隐蔽性**：地面看起来安全的区域，地下可能已经充满丧尸
- **出口不可预测**：丧尸可能从任何一个排水口、窨井、雨水篦子爬出

**游戏设计推论**：
- 地下车库、商场B1/B2层、地势低洼的小区在一周后都会成为高危区——不是因为有丧尸从地面进入，而是从排水系统上来的
- 玩家在街头听到的窨井盖下面有刮擦声——那不是老鼠，是丧尸在暗渠里经过

#### 第三层：综合管廊/电缆隧道（有限但关键）

上海等城市有大规模的地下综合管廊（电力、通信、给水共用通道）。这些管廊断面足够通行，连接多个街区。其中电力管廊内部干燥、温度稳定——不像排水管那样潮湿——可能成为丧尸的"舒适通道"。

但管廊的入口有限（主要在变电所/泵站），且内部有防火分区（防火门可以阻挡丧尸），所以不是主要扩散路径，而是**局部连接通道**。

### 13.4 三角洲破裂：按天推演

#### 6/28（爆发日）——多点开花

```
状态：长江沿线各城市几乎同时出现失控事件
交通：高铁/高速仍部分运行，大量市民试图撤离
政府：各城市发布封锁令，但封锁线内部已经在崩
关键事件：上海-苏州地铁连接线将两座城市的爆发点贯通
```

- 晚18:00 前，长江三角各市的主要医院已全部超负荷或被冲击
- 晚20:00，南京、上海、杭州几乎同时请求中央增援
- 晚22:00，各市高速入口严重拥堵——且堵死的车流中不断出现车内咬人事件
- 午夜前后，地铁系统基本停运，但**隧道内的丧尸已经出不来了**

#### 6/29（死亡走廊形成）——三线贯通

```
沪宁走廊：    上海 →（G42/11号线/S1线）→ 苏州 →（G42/京沪高铁）→ 无锡 → 常州 → 南京
沪杭走廊：    上海 →（G60/沪杭高铁）→ 嘉兴 → 杭州
沿江走廊：    南京 →（G42+长江）→ 镇江 → 扬州
```

- **白天**：地面丧尸密度中等，主要集中在主干道和水道两侧
- **下午~夜间**：第一天夜间丧尸全面活跃，沿街行走的丧尸形成第一波"夜间流"
- **关键点**：高架/高速的堵死车流中，大量尸体在车内或车旁开始尸变 → **高架从"逃生通道"变成"死亡生产线"**

**为什么高架/高速瘫痪后反而更危险**：
1. 拥堵导致大量车辆停在原地 → 车内的人要么感染了、要么暴露了
2. 堵车→人下车→人变成丧尸→丧尸在车缝中游荡
3. 高架路面干燥开阔，但高架**下方**的桥墩区域潮湿阴暗 → 丧尸从高架上掉下去或爬下去，在桥墩下聚集
4. 最终结果：高架/高速的**路面密度低，但上下匝道和桥墩区域密度极高**

#### 6/30 ~ 7/1（地铁溢出）——拐点到来

地铁停运后48小时左右，隧道内已经积累了大量的丧尸。它们沿着隧道持续行走，到达离地面最近的车站出入口、风井、检修通道。

- **6/30 夜间**：第一个"地铁溢出"事件——某地铁站出入口的卷帘门被隧道内丧尸群推开，大量丧尸从站内涌出
- **7/1 凌晨~上午**：连锁反应。全市各站陆续溢出
- **溢出效应**：原本相对安全的地面区域（尤其是地铁站周边1km范围内），在数小时内丧尸密度骤增

> **这对玩家意味着什么**：6/29~6/30，地铁站还是一个"可以冒险进入"的地点（隧道的丧尸密度高但站台尚可通行）。7/1之后，地铁站成为高危区——不是站台本身危险，是隧道里随时可能有丧尸群涌出。

#### 7/2 ~ 7/7（饱和扩散）——稳态前的混乱

- **跨市贯通完成**：沪宁、沪杭、沿江三条死亡走廊全线连通
- **大运河走廊成形**：镇江—常州—无锡—苏州—嘉兴—杭州，沿运河连续高密度带
- **太湖流域面状覆盖**：环太湖各城镇之间的乡镇、村庄、农田中的丧尸密度趋同
- **崇明岛/长兴岛/横沙岛**：渡轮停运前有零星的感染输入，但岛上的丧尸密度远低于市区——**长三角最有潜力的安全区**
- **杭州湾南岸（宁波/绍兴/舟山）**：相对独立的水系（甬江/曹娥江/钱塘江），南部山区多，扩散较慢

```
                      长江（天然屏障，北岸相对安全）
                          |
```

### 13.5 长江——最关键的天然边界

长江在长三角段（江阴以下）宽度从2km逐渐展宽至入海口的20-90km。这条大河在物理上把长三角分为**江南**和**江北**两个世界。

**跨江通道（从上游到下游）：**

| 通道 | 类型 | 6/30后状态 | 战略意义 |
|------|------|-----------|---------|
| **南京长江大桥/二桥/三桥/四桥** | 公路+铁路桥 | 全部堵塞或失守 | 南京南北通道全部切断 |
| **润扬大桥（镇江-扬州）** | 公路桥 | 堵死 | 苏南→苏北的主要通道之一 |
| **江阴大桥** | 公路桥 | 堵死或半堵死 | 无锡/泰州之间 |
| **苏通大桥（苏州-南通）** | 公路桥（主跨1088m） | 严重堵塞，但可能有军事管制段 | 苏南→苏北最大通道 |
| **崇启大桥（上海崇明-启东）** | 公路桥 | 可能由军队控制 | 上海向北的唯一陆地通道 |
| **上海长江隧桥（浦东-长兴-崇明）** | 隧道+桥 | 隧道被堵死，桥面可能由军队控制 | 崇明岛连接到浦东的唯一通道 |

**核心结论**：
- 长江是一条有效的屏障——不是因为丧尸过不了江（桥上有丧尸），而是因为江北的感染输入**延迟且有限**
- **苏北（南通、泰州、盐城、淮安）是长三角残余秩序的主要退守方向**
- 中央专家组在南京无法前进后，最现实的方案是绕道北岸——经苏北从崇启大桥进入崇明岛，再图入沪

### 13.7 对现有游戏系统的映射建议

当前 `core.js` 中的 `chasedByZombies`（0-5）是一个位置无关的全局值。建议未来扩展时考虑：

**按区域的基准密度参数**（不动现有架构，只作为场景设计参考）：

### 13.8 对游戏剧情的结构性影响

1. **游戏前期（东明街道）**：玩家在相对有限的区域内探索，学习四定律，理解"近水/远水""地上/地下""白天/夜晚"的风险差异。

2. **游戏中期（出城抉择）**：玩家必须选择一个方向突围。三条出城路线对应三种游戏体验——A（挑战难度，正面穿越高密度区）、B（探索难度，绕行野外）、D（速通难度，赌地铁隧道）。E结局走B路线最合理。

3. **游戏后期（三角洲漫游）**：如果游戏范围扩大到长三角，每个区域的状态差异提供了丰富的叙事支点——崇明岛的军事哨站、太湖西岸的幸存者社区、杭州湾南岸的相对秩序、苏北的政府残余。

4. **真相线的地理逻辑**：走申嘉湖高速从湖州绕到芜湖，沿途经过的乡镇密度渐低，是最可行的地面路径。这条路线本身就可以是一段独立的"公路生存"剧情链。

5. **E结局的悲壮感补充**：当玩家终于到达芜湖融汇化工，打开核心废料区的开关，引发最后一波大规模泄露时——触发泄露的**震动和声波**可能沿着地下管廊和地下水系统传导，引爆长三角死亡区大面积连锁尸潮反应。这发生在真相大白的那一刻，城市的远方传来一阵阵低沉的轰鸣。<br><br>这不是玩家能看到的——但如果玩家活着走出工厂，站在高处望向东方，会看到地平线上扬起的烟尘。那是上海方向。
